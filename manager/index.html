Para Equipos y PMs

Claude Code para managers de IPA Colombia

Nota

Esta ruta de la guía recoge el trabajo de Juan Felipe García, Country Director de IPA Colombia, quien construyó un workspace personal con Claude Code durante 3 meses y documentó las herramientas y plantillas que creó. Este repo es el sitio oficial; un mirror técnico (con la misma fuente) vive en Quintero-DiegoIPA/ipa-claude-code-intro.

Qué es esto

IPA nos asignó licencias Enterprise de Claude, incluyendo acceso a Claude Code. Esta sección documenta cómo un manager puede construir un workspace con Claude Code, y te da herramientas para que explores si te interesa.

Encontrarás:

  • Cómo instalar Claude Code paso a paso
  • Qué puedes y qué no puedes hacer según la política de IPA (ver Seguridad)
  • Módulos que explican cada herramienta: qué problema resuelve, la filosofía detrás, cómo está construida
  • Plantillas funcionales que puedes copiar a tu propio workspace
  • Ejemplos reales de proyectos (plan estratégico, análisis de costo-efectividad)

Qué es Claude Code

Claude Code es una forma de conversar con Claude directamente desde tu computador. Hay una app de escritorio, una versión de terminal y una versión web. A diferencia de claude.ai (el chat web normal), Claude Code puede:

  • Leer y editar archivos en tu computador (documentos, Excel, carpetas)
  • Generar documentos (Word, PowerPoint, Excel) listos para compartir
  • Buscar en tu correo (con integración de Gmail)1
  • Automatizar tareas repetitivas (reportes, seguimiento, formateo)
  • Recordar quién eres y cómo trabajas entre sesiones, mediante un archivo de contexto llamado CLAUDE.md
Tip¿Necesito saber programar?

No. La app de escritorio tiene una interfaz visual donde conversas con Claude igual que en un chat. La diferencia es que Claude puede acceder a tus archivos locales y generar documentos. Si ya has usado Claude o ChatGPT en el navegador, Claude Code es el siguiente paso.

ImportanteLo que NO es

No reemplaza tu criterio. No es almacenamiento. No es un motor de búsqueda. Siempre revisa lo que produce.

Nota¿Usas Box para tus archivos?

Algunos módulos (como el Knowledge Base y el Plan Estratégico) leen archivos desde carpetas de Box. Para que funcione, necesitas tener Box Drive instalado y sincronizado en tu computador. Si no lo tienes, descárgalo desde box.com/drive o pide ayuda al helpdesk.

Si Claude va a tener acceso a tu Box, considera instalar también el guardrail de Box. Limita a Claude a leer solo las carpetas que tú autorizas, en vez de cualquier archivo de tu Box. Toma 5–10 minutos configurarlo.

Por dónde empezar

Hay un paso a paso completo en Cómo empezar. Si quieres ir directo a algo concreto:

1. Instalar

Verifica tu licencia e instala Claude Code en 10–15 minutos.

Ir a instalación →

2. Conocer los límites

Qué puedes y qué no puedes hacer con la licencia Enterprise.

Ver Seguridad →

3. Cargar un starter pack

Setup completo por rol (growth, revisiones, people, procurement). 10–20 min cada uno.

Ver starter packs →

4. Aprender por capacidad

Si prefieres entender qué se puede hacer y armar tu setup a medida.

Ver módulos →

TipStarter pack o módulos: ¿cuál es cuál?

Starter packs son setups listos por rol (managers de growth, weekly review, people, procurement). Cargas uno y arrancas con estructura validada. Módulos son explicaciones por capacidad (comunicaciones, gestión de tareas, KB). Útil si todavía no tienes claro qué empezar a hacer y prefieres entender primero. Las dos rutas son legítimas, puedes empezar por una y migrar.

No hay presión

No necesitas adoptar todo. No necesitas saber programar. No necesitas construir nada complejo.

Revisa lo que te llame la atención, hazte preguntas, y si algo te interesa, pruébalo.

Notas

  1. La integración con Gmail requiere configurar un conector adicional (MCP server) y autorización previa. No viene activada por defecto. Consulta con tu líder de área antes de configurarla.↩︎
