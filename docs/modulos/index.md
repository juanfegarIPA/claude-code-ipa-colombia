# Módulos — Qué se puede construir con Claude Code

Cada módulo describe una capacidad del workspace del Country Director de IPA Colombia. Para cada uno encontrarás: qué problema resuelve, la filosofía detrás, cómo está construido, un ejemplo real, y cómo empezar.

No necesitas todos. Revisa los que te llamen la atención y empieza por el que más se conecte con tu trabajo.
